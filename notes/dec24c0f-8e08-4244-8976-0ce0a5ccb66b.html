Nested Strong Params

params.permit(:site_name, [data: {object: {metadata: [:client_id]}}])

This will allow params that look like this:

{
  "site_name": "some site",
  "data": {
    "object": {
      "metadata": {
        "client_id": "some id"
      }
    }
  }
}
