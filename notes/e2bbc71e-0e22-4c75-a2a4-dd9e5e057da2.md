You can use `NULLIF` to safely divide in `SQL`:

```sql
(count(1)::numeric / NULLIF((
          SELECT
            count(1)
          FROM
            playlists
          WHERE
            state = 'published'), 0)
```

When these values match (`NULLIF(0, 0)`), null will be returned instead of dividing by 0.
